The bin directory contains the code for performing the HS capture CNV analysis. The code should be executed
in the following order:

1 - 01_mDiGS_coverage_normalization
2 - 02_mDiGS_CNV_calling
3 - 03_mDiGS_CNV_analysis

More details about the executing the code at each step is located in the READ_ME.txt files within each bin subdirectory.
